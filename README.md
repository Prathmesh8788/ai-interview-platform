# 🚀 AI Interview Platform

An AI-powered interview preparation platform that simulates real interview scenarios using intelligent question generation and evaluation. This project helps users practice interviews interactively and improve their performance with AI-driven feedback.

---

## 📌 Features

- 🎤 AI-based mock interview system  
- 🤖 Dynamic question generation using AI  
- 🧠 Real-time answer evaluation  
- 📊 Performance feedback and scoring  
- 🔐 User authentication system  
- 💾 Data storage with database integration  

---

## 🛠️ Tech Stack

**Frontend:**
- React.js  
- HTML5, CSS3  

**Backend:**
- Python  
- Django / FastAPI  

**Database:**
- PostgreSQL  

**AI Integration:**
- Gemini AI API  

---

## ⚙️ Installation & Setup

### 1. Clone the repository
```bash
git clone https://github.com/Prathmesh8788/ai-interview-platform.git
cd ai-interview-platform
```

### 2. Backend Setup
```bash
cd backend
python -m venv venv
venv\Scripts\activate   # Windows
pip install -r requirements.txt
```

### 3. Run Backend Server
```bash
python manage.py runserver
```

### 4. Frontend Setup
```bash
cd frontend
npm install
npm start
```

## 📡 API Endpoints

| Method | Endpoint | Description |
|--------|--------|-------------|
| POST | /api/interview/start | Start interview |
| POST | /api/interview/answer | Submit answer |
| GET | /api/interview/result | Get feedback |

---

## 🧠 How It Works

1. User logs in and selects interview type  
2. AI generates questions dynamically  
3. User submits answers  
4. AI evaluates responses and provides feedback  
5. Performance is stored and displayed
